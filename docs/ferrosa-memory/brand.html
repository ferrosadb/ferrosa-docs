Ferrosa Memory Brand Guidelines

Visual identity for Ferrosa Memory — the typed, durable memory layer of the Ferrosa platform. Use these assets and guidelines for the operator workbench, documentation, presentations, and community materials.

Logo

Ferrosa Memory carries the Ferrosa family grammar — a serif chemistry monogram inside a periodic-table frame with an orbital ring — but swaps "Fe / 26" for "Fm / 100" (Fermium) and adds a memory-graph motif: nodes linked by retrieval paths. The mark renders in electric blued steel, signalling a distinct subsystem while staying unmistakably part of Ferrosa.

Voice & Tone

Ferrosa Memory speaks to operators and engineers debugging what an agent remembers. Confident and direct, never mystical about "AI memory." We respect the reader's intelligence.

{ }

Technical

Lead with specifics. "Hybrid BM25 + vector recall with LLM rerank," not "smart memory."

→

Direct

Short sentences. Active voice. State what a tool does, then what it returns.

≡

Honest

Forgetting has blast radius. We show the edges that break before anything is deleted.

□

Builder-First

Written for engineers inspecting and curating the knowledge base, not executives reading a deck.

Logo Usage

Do

Use the full logo on deep-field backgrounds (#05070c to #0e1726). Maintain clear space equal to the height of the "Fm" mark on all sides. Use the icon mark alone when space is tight (favicons, avatars, badges).

Don't

Don't place the logo on busy or light surfaces without inverting. Don't stretch, rotate, or recolor the gradient to warm tones. Don't reuse the base Ferrosa "Fe" terracotta mark for Memory.

Minimum Size

Full logo: 140px wide minimum. Icon mark: 24px minimum. Favicon: 16px (use favicon.svg, which is optimized for small sizes).

Co-Branding

When shown alongside base Ferrosa or other family marks, maintain equal visual weight. Separate with a 1px vertical divider (var(--border)) and 24px horizontal padding.

Naming

Correct

Ferrosa Memory (two words, both capitalized). Lowercase ferrosa-memory in code contexts (repo, crates, package names). Crates use the ferrosa-memory- prefix. The MCP tool surface is fmem.

Incorrect

FERROSA MEMORY (all-caps), FerrosaMemory (one word), Ferrosa-Memory (title-case hyphen), "ferrosa mem," or "FM" as a standalone product name. Never imply it is a separate product from Ferrosa.
