Design System

Accessible, responsive components built on the Ferrosa Memory brand tokens — electric blued steel on a deep field. Core Blue is the single action color; Electric Cyan is reserved for highlights, code, and data accents. Every example meets WCAG 2.1 AA contrast in both themes and works with a keyboard. Toggle dark mode in the header. Use the Copy button on any example to grab the markup.

Typography

Inter for UI, JetBrains Mono for identifiers, scores, and tool output. Georgia serves the Fm monogram only — never body copy. The scale is fluid and capped for readability (body measure 65ch).

Spacing & radius

A 4px spacing base (--fm-space-1…8) and one radius scale (xs 4 / sm 8 / md 12 / lg 16). Pick a step and stay on it; do not mix radii within a surface.

Elevation & motion

Three shadow steps tinted to the deep-field ink, never pure black. Motion is short (160ms) and eased, and is disabled entirely under reduced-motion.

Buttons

One primary action per view. Labels are 1–3 words and never wrap. All variants meet AA contrast and have hover, active, focus, disabled, and loading states. Danger is reserved for destructive, blast-radius actions like hard forget.

Menu

A button toggles a menu with aria-expanded and aria-haspopup. It closes on Escape or an outside click and returns focus to the button. Destructive items sit below a separator, in danger red.

  </div>
</div>

Drawer & navigation

An off-canvas panel for navigation on small screens. It traps focus, closes on Escape or scrim click, and restores focus. Inside is the standard nav list with an active item.

Accordion

Built on native <details>, so it is keyboard- and screen-reader-accessible with no JavaScript. Each panel toggles independently.

What is a fold?
A consolidated summary of a trajectory — session entities compressed into a durable, retrievable fact.

Candidates from BM25, ANN, phonetic, and the Datalog frontier are merged with Reciprocal Rank Fusion, then optionally reranked by the judge model.
Is forgetting reversible?
Default retract mode is reversible via restore. Hard mode is permanent and always shows the blast radius first.
