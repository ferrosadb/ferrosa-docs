A memory server, not a memory vibe.

Most agent memory products stop at text chunks and vector similarity. Ferrosa Memory treats memory as operational data that can be linked, queried, corrected, explained, and removed.

fact

Bi-temporal knowledge

Facts keep valid-time and transaction-time history, so agents can ask what is true now and what changed earlier.

graph

Entity relationships

People, projects, skills, sessions, documents, and tools are connected with typed edges instead of buried in prose.

recall

Hybrid retrieval

Recall blends lexical, vector, phonetic, graph, workspace, recency, and feedback signals before returning evidence.

forget

Auditable forgetting

Preview the blast radius, retract reversibly, or hard-delete with an append-only journal for trust and compliance work.

mcp

Agent-native tools

Expose a focused default toolset for daily work, with advanced graph, fold, Datalog, and operator tools when needed.
